What is this site and how do I use it?

This tool allows users to upload numerical data to check if it obeys Benford's law (aka the first-digit law).
In brief, it is an observation that in many real-life sets of numerical data, the leading digit is likely to be small. You can upload your own data to validate it against this law (and choose to save it to a database) and view existing data in the database.

Upload and Validate Data

You can uplaod your own data (as a .csv file) to check if your data obeys Benford's law. You can also choose to save it to a database.

View Data

Here you can see the data that has been uploaded previously.

Each time a file is uploaded and saved to the database, a parent record (with basic info like filename, date, a unique identifier, etc.) is created; these are Ingestion Jobs. The records (the numerical values) are saved to Ingested Data.
